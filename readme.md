Useful less mixin

Installation
---

```sh
npm i less-mixin
```

Usage
---

```less
@import "mixin/all";
```

Mixin
---

- `.clearfix`
- `.size`
- `.font-size`
- `.color`
- `.color-reverse`
- `.opacity`
